Somebody鼠标键盘外设、视频设置、准心CFG参数

2019-05-24 分类：选手外设/设置 阅读(15862) 评论(0)

以下是中国队天禄（Tyloo）俱乐部CSGO选手Haowen “somebody” Xu（外号小鬼、鬼王爷）所使用的硬件外设、视频画面设置以及准心（准星）、视角指令参数等：

配件外设

  • 显示器：明基XL2546（购买地址）
  • 鼠标：卓威EC2-A（购买地址）
  • 鼠标垫：卓威G-SR（购买地址）
  • 键盘：XTRFY K2-RGB（购买地址）
  • 耳机：HyperX Cloud Flight（购买地址）
  • 显卡：英伟达GEFORCE RTX 2080 Ti（购买地址）

鼠标设置

  • DPI：400
  • 灵敏度：2.75
  • eDPI：1100
  • HZ：1000
  • 开镜时灵敏度：1.00
  • 鼠标加速：关
  • 系统灵敏度：5
  • 原始输入：开

视频设置

  • 画面分辨率：1280×960
  • 宽高比：4:3
  • 缩放模式：拉伸
  • HZ：240

准心（准星）参数

cl_crosshairalpha 255; cl_crosshaircolor 1; cl_crosshaircolor_b 255; cl_crosshaircolor_r 255; cl_crosshaircolor_g 255; cl_crosshairdot 0; cl_crosshairgap -2; cl_crosshairsize 3.5; cl_crosshairstyle 1; cl_crosshairusealpha 1; cl_crosshairthickness 1; cl_crosshair_drawoutline 0; cl_fixedcrosshairgap -4;

视角设置

viewmodel_fov 68; viewmodel_offset_x 2.5; viewmodel_offset_y 0; viewmodel_offset_z -1.5; viewmodel_presetpos 3; cl_viewmodel_shift_left_amt 1.5; cl_viewmodel_shift_right_amt 0.75; viewmodel_recoil 1; cl_righthand 1;
