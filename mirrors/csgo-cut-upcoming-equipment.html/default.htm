盘点CSGO曾打算加入的游戏内容（武器、地图、模式等）

2019-06-16 分类：CSGO新闻资讯 阅读(5864) 评论(0)

本文收集了反恐精英：全球攻势开发团队曾打算为CSGO添加一大堆游戏内容，包括新武器、地图和功能等等，这些内容曾随着游戏更新加入到文件里被玩家挖掘出，但由某些原因被抛弃或推迟。或许哪一天，这些内容会正式发布。

*以MP5-SD为例，这把微冲于2018年8月15日加入到游戏里，但在更早的一年多前CSGO团队就将相关文件更新进游戏里。

 

武器

SCAR-17

SCAR-17取自现实中的特种部队战斗突击步枪，其名称来自于SCAR重型版本（Heavy，SCAR-H，Mk 17 Mod 0）的组合，使用7.62x51mm北约弹药及FN FAL的20发弹匣。

早在2011年的CSGO Beta版里，SCAR-17就存在过，武器代码“weapon_scar17”，拥有完整的模型和贴图，并且能够使用，不知何原因，在CSGO正式发布后SCAR-17被移除，最终只存在于CSGO图标中。

如果你想体验Scar-17，可以下载CSGO远古Beta版。

 

TAC-21

TAC-21是一把栓动式狙击枪，使用.308子弹，也有其他型号和AWP一样使用.338子弹。

在CSGO中，曾多次更新了这把武器的相关文件，比如2014年10月22日首次更新了这把武器的开火声音引用文件，2016年1月20日更新了相关动作文件，只不过这类数据在之后的更新中都被删除了，武器代码“weapon_tac21”。

图片来自网络，玩家自制模型

 

M48战斧（M48 Tomahawk）

M48战斧是一把轻型近战武器，顶部附有刀刃，适合随身携带。

在2014年7月1日的更新中，加入了这把武器模型贴图引用VMT文件，从文件路径来看，M48战斧被归纳为匕首，可作为近战武器使用。

materials/models/weapons/v_models/knife_m48_tomahawk/knife_m48_tomahawk.vmt

 

Canis匕首和Cord匕首

这两把武器是和MP5-SD一起在2017年3月15日更新了武器图标，虽然MP5-SD正式更新了，但这两把匕首还未有下文。

 

双管猎枪（Double Barreled Shotgun）

双管猎枪是一把可以快速射击双发子弹的霰弹枪。

在CSGO中，曾在give_all_items.cfg脚本文件中出现武器代码“weapon_dbarrel”。

图片来自网络

 

复合弓（Compound Bow）

复合弓最初在2016年5月31日更新中加入了相关代码，根据文件提供的信息，复合弓需要按住按键拉钩，松开按键发射。弓箭可以和投掷物组合，包括手榴弹、燃烧弹、烟雾弹、闪光弹等，将其射得更远。此外，和现有的枪械不同，弓箭有飞行时间。

*在2016年5月31日更新还包含了遥控C4代码，如今已正式加入到头号特训模式。

 

自定义密码C4（Custom Code C4）和无线电C4（Radio C4）

自定义密码C4和无线电C4曾在give_all_items.cfg脚本文件中提到，武器代码分别为“weapon_c4_customcode”和“weapon_c4_radio”，前者能够自定义下包密码（现有C4密码为7355608），后者可能是能够远距离引爆C4。

 

捕兽夹（Bear trap）和锅铲（Spatula）

在CSGO 2018年7月6日更新中发掘了捕兽夹和锅铲的武器图标，一同更新的还有锤子、扳手和斧头，后3件物品被作为头号特训近战武器使用，而锅铲可能是觉得太恶搞给移除了。至于捕兽夹，因为是作为陷阱使用（头号特训地图有大片草地），或许在未来会更新。

 

地图

在2011-2012年的CSGO Beta测试期间，玩家就发现了不少未发布的地图，其中有的似乎制作完成，有的只制作了部分模型，但这些地图至今仍未有消息。

de_alleyway

de_alleyway是一张拆弹地图，背景设定在深夜的小巷中，CT要阻止T破坏油罐车，另外地图包含了油罐车被炸毁的模型，或许有爆炸动画。

 

de_depot

de_depot是一张拆弹地图，背景设定在俄罗斯火箭发射基地里。

 

de_embassy

de_embassy是一张拆弹地图，背景设定在英国大使馆。

 

de_balkan

de_balkan是一张拆弹地图，由反恐精英:零点行动官方地图Vostok重制而来（或许是当时V社希望吸引老玩家玩CSGO）。

 

de_cranes

de_cranes是一张拆弹地图，背景似乎设定在由起重机的工地，该地图只保留了雷达文件。

其他

其他还有不少地图出现在CSGO中，但信息过少只知道地图名和模式：

  • ar_baloney，军备竞赛
  • de_glass，拆弹模式

 

人物/游戏模式

格鲁吉亚防暴警察（Georgian Riot Police）

和其他人物设计稿不同，格鲁吉亚防暴警察拥有完整的人物模型和动作文件，玩家可以在早期Beta版玩到。

 

清道夫模式（Scavenger Mode）

清道夫模式是让空手玩家在地图上找到武器并淘汰对方队伍赢得回合，当获得一定数量胜利回合后则赢得比赛。

虽然这个模式还未推出，但CSGO团队加入了该模式的成就系统：

开胃小菜/Smorgasbord

  • 在清道夫模式中，在单个回合内将每—类可用的枪械都使用—次
  • Use every available weapon type in a single round in Scavenger Mode

屹立不倒/I’m Still Standing

  • 在清道夫模式中，以最后活着的玩家身份赢得回合胜利
  • Win the round as the last man standing in Scavenger Mode

活在死亡边缘/Living on the Edge

  • 在清道夫模式中进行—次刀杀
  • Get a knife kill in Scavenger Mode

 

抢劫模式（Heist Mode）

抢劫模式或许是仿照游戏《收获日》设计，最初在2016年5月31日更新加入了相关代码，其中“func_bomb_target”代码用于让玩家将C4安放在墙上，还有配备了挎包的无贴图模式专属人物角色“Tm_heavy_riot”，以及求饶动作。

 

Tilegen模式

暂无此模式更多信息，只知道2016年5月31日更新中加入了相关代码。

 

功能

自定义HUD

在CSGO Beta测试中，玩家可以自定义HUD功能，比如修改雷达显示位置、字体大小颜色等等。不过由于自定义HUD存在严重漏洞，CSGO团队禁用了该功能，并表示会在之后回归。然而到了2018年，CSGO团队成员John McDonald却表示没有回归计划，缘由是要保障玩家们的竞技公平环境。

 

CSGO商店（CS: GO Store）和人物装扮系统/角色皮肤（Wearables）

不同于现有的游戏内购买系统，CSGO团队曾打算推出能让玩家随心购物的功能CS: GO Store，在2016年野火大行动更新中加入了部分代码，包括购物车、精选商品、特价活动，这些我们都可以通过翻译文本查看到。

接上，商店功能其中就打算出售角色皮肤商品，分CT和T阵营，角色饰品包括脸部、帽子、上衣、眼睛、面罩、手套和裤子，让玩家能够随心搭配角色外观。在之后，除了手套饰品正式加入到游戏里，其他外观饰品没有了下文，或许是担心影响到游戏竞技性。

*早在CSGO推出前，开发计划的设计稿就包括了外观饰品：

 

挂件/钥匙链（Key Chain）

在2017年6月20日的CSGO更新中，开发团队在游戏文件里加入了挂件/钥匙链模型引用代码（仅包括代码，模型、贴图并未更新），后来没了下文。
