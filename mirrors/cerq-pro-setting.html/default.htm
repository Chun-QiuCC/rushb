CeRq鼠标键盘外设、视频设置、准心CFG参数

2019-11-08 分类：CSGO新闻资讯 / 选手外设/设置 阅读(7678) 评论(0)

以下是Evil Geniuses俱乐部CSGO职业选手Håvard “rain” Nygaard所使用的鼠标、键盘、显示器、耳机等硬件外设、以及视频画面设置、准心（准星）和视角控制台指令参数等：

配件外设

  • 显示器：ZOWIE GEAR 卓威奇亚 XL2540（购买地址）
  • 鼠标：ZOWIE GEAR 卓威 奇亚 EC2鼠标（购买地址）
  • 鼠标垫：ZOWIE GEAR 卓威 奇亚 G-SR 电竞鼠标垫 （购买地址）
  • 键盘：雷蛇(Razer)黑寡妇蜘蛛（购买地址）
  • 耳机：森海塞尔（Sennheiser） GAME ZERO （购买地址）
  • 显卡：华硕 GTX 1080 TI （购买地址）

鼠标设置

  • DPI：400
  • 灵敏度：2.20
  • eDPI：880
  • HZ：1000
  • 开镜时灵敏度：0.70
  • 鼠标加速：关
  • 系统灵敏度：6
  • 原始输入：开

视频设置

  • 画面分辨率：1024×768
  • 宽高比：4:3
  • 缩放模式：黑边
  • HZ：240

准心（准星）参数

cl_crosshairstyle 1; cl_fixedcrosshairgap -6; cl_crosshaircolor 5; hud_showtargetid 0; cl_crosshair_friendly_warning 0; cl_crosshair_sniper_width 1;

视角设置

viewmodel_fov 68; viewmodel_offset_x 2.5; viewmodel_offset_y 2; viewmodel_offset_z -2; viewmodel_presetpos 0; cl_viewmodel_shift_left_amt 1.5; cl_viewmodel_shift_right_amt 0.75; viewmodel_recoil 0; cl_righthand 1;

视角（手臂/武器）晃动

cl_bob_lower_amt 5; cl_bobamt_lat 0.1; cl_bobamt_vert 0.1; cl_bobcycle 0.98;
