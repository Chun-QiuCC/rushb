CSGO DEMO录制、下载、保存文件夹路径、回放

2020-02-24 分类：CSGO指令命令 / CSGO教程攻略 阅读(70179) 评论(0)

在CSGO中，玩家可以自己录制DEMO文件，所谓DEMO，就是游戏专用的录像文件，玩家可以通过CSGO客户端，回放游戏过程，方便玩家录制加工成精彩集锦视频，或是判断玩家是否作弊。同理，玩家可以从赛事网站比如HLTV下载比赛GOTV DEMO重新观看比赛，亦或者将DEMO文件复制分享给好友。

DEMO文件的后缀名为“.dem”

本教程的录制DEMO仅为POV DEMO/个人DEMO，即仅包括个人视角，如果你想录制全局DEMO（包含所有人视角），》请看这个教程《。

下载官方匹配竞技模式的DEMO文件、保存路径

CSGO中，官方模式默认只有竞技模式会自动帮玩家录制保存DEMO文件，至于其他模式或是社区服，如果玩家想保存DEMO，需要自己手动启用录制功能，教程见下方。

首先是如何下载官匹竞技模式DEMO文件：

打开游戏——“观看匹配比赛和锦标赛”——“我的比赛”，选择比赛，点击右下角的“下载”按钮即可。

注意：

建议开加速器下载。

官方服务器似乎只帮你保存DEMO一星期时间，如果超过这个时间，那么比赛则无法再下载DEMO。

下载完成后，在“观看匹配比赛和锦标赛”——“已下载”便可查看保存的比赛DEMO，点击右下角的“观战”即可回放比赛DEMO。

自己录制DEMO文件、及保存路径

本教程的录制DEMO仅为POV DEMO/个人DEMO，即仅包括个人视角，如果你想录制全局DEMO（包含所有人视角），》请看这个教程《。

上面说到，官方模式只有竞技模式会自动录制DEMO，其他模式或者是娱乐社区服，需要自己手动录制。

进入游戏后，打开控制台，输入DEMO录制控制台指令：

record [文件名]

输入后，控制台会提示“Recording to XXXXX.dem…”，即录制到XXXXX.dem文件

*开启录制功能可能会降低游戏性能。

当你觉得差不多时，便可以输入停止录制命令：

stop

输入后控制台提示“Completed demo, recording time XX, game frames XXXX.”，即DEMO录制完成，录制了XX时间，比赛大小XXXX。

和上面不同，自己录制的DEMO文件实在上一级目录，即“\steamapps\common\Counter-Strike Global Offensive\csgo”内：

同理，自己下载的DEMO文件也可以放在这个文件夹内，比如HTLV下载的GOTV比赛DEMO。

 

回放DEMO

自己录制或是网站下载的DEMO可通过游戏客户端回放，记得DEMO文件要放在CSGO根目录里，并且使“.dem”格式

这里以HLTV下载的GOTV比赛DEMO为例，已经放到CSGO根目录：

打开游戏，控制台输入DEMO界面指令：

demoui

便会弹出DEMO控制界面，点击“Load”，打开选择的DEMO文件，便会进入DEMO回放。

另外DEMO控制界面可以快放和慢放，或者直接进入下一回合，具体请自行尝试。
