HObbit鼠标键盘外设、视频设置、准心CFG参数

2021-05-10 分类：选手外设/设置 阅读(8731) 评论(0)

以下是Gambit俱乐部CSGO职业选手Abay “HObbit” Khasenov所使用的硬件外设、视频画面设置以及准心（准星）、视角指令参数等：

配件外设

  • 显示器：卓威 奇亚 ZOWIE GEAR  XL2540（购买地址）
  • 鼠标：VAXEE OUTSET AX
  • 鼠标垫：赛睿 (SteelSeries) QcK Heavy（购买地址）（领取优惠券）
  • 键盘：XTRFY K2（K4购买地址）
  • 耳机：HyperX CLOUD II（购买地址）
  • 显卡：NVIDIA GEFORCE RTX 2080

鼠标设置

  • DPI：800
  • 灵敏度：1.10
  • eDPI：880
  • HZ：1000
  • 开镜时灵敏度：0.75
  • 鼠标加速：0
  • 系统灵敏度：6
  • 原始输入：开

视频设置

  • 画面分辨率：1680×1050
  • 宽高比：16:10
  • 缩放模式：拉伸
  • HZ：240

准心（准星）参数

cl_crosshairsize 3; cl_crosshairgap -2; cl_crosshairthickness 0.4; cl_crosshaircolor 5; cl_crosshaircolor_b 144; cl_crosshaircolor_g 195; cl_crosshaircolor_r 27; cl_crosshairdot 1; cl_crosshairstyle 4; cl_crosshairalpha 250; cl_crosshairgap_useweaponvalue 0; cl_crosshairusealpha 1; cl_crosshair_drawoutline 0; cl_crosshair_dynamic_maxdist_splitratio 0.35; cl_crosshair_dynamic_splitalpha_innermod 1; cl_crosshair_dynamic_splitalpha_outermod 0.5; cl_crosshair_dynamic_splitdist 7;

视角设置

viewmodel_fov 68; viewmodel_offset_x 2.5; viewmodel_offset_y 0; viewmodel_offset_z -1.5; viewmodel_presetpos 3; cl_viewmodel_shift_left_amt 1.5; cl_viewmodel_shift_right_amt 0.75; viewmodel_recoil 0; cl_righthand 1;

视角（手臂/武器）晃动

cl_bob_lower_amt 21; cl_bobamt_lat 0.4; cl_bobamt_vert 0.25; cl_bobcycle 0.98;
