查看CSGO举报记录以及封禁结果

2019-03-17 分类：CSGO教程攻略 阅读(60435) 评论(0)

CSGO中有时会遇到疑似作弊/开挂玩家，利用游戏内举报功能我们可以将他们提交给V社。不过V社并不一定将具体结果反馈给玩家，需要玩家自己手动查询。以下是查看CSGO举报历史记录，以及批量查询被举报玩家是否封禁详细教程。

通过Steam提供的CSGO个人游戏数据页面，我们可以查询到游戏中举报玩家列表，包括时间、受举报玩家以及举报详情。

CSGO个人游戏数据页面地址：

https://steamcommunity.com/my/gcpd/730/?tab=playerreports

*原本此页面可以查询至今所有被举报玩家，不过2019年后Steam更新了相关条款，玩家只能查询2017年11月以后的记录

点击被举报玩家头像，就可以进入他们的Steam个人资料页面，如果页面右边栏出现“记录在案的VAC封禁”或“记录在案的游戏封禁”，那么就表示这名玩家已被封禁。

*虽然其他游戏也有使用VAC，但如果玩家被VAC基本上可以认为是CSGO封禁；如果CSGO玩家被游戏封禁，那么可能是监管封禁，也有可能是其他游戏封禁（比如吃鸡）

以上就是基本的CSGO举报记录以及结果查询方法，如果你想要批量查询所有被举报玩家的封禁结果，可以继续看以下内容：

 

配合浏览器扩展插件Ban Checker for Steam，我们可以批量查询所有被举报玩家。

1、在浏览器上安装插件/扩展：Ban Checker for Steam

Chrome下载地址；Firefox下载地址

*若Chrome无法访问，可从百度网盘下载，提取码: pbhi

*如果你之前已经安装过此插件，请确保是最新版1.2.2.1。

插件下载完成后并启用，进入下一步骤。

 

2、注册Steam API 密匙（Key）

*如果不注册密匙，则无法查询所有玩家

先进入Steam API 密钥（Key）注册页面：https://steamcommunity.com/dev/apikey

如图，输入“域名名称”（随意起），勾选“I agree to the Steam Web API Terms of Use”，点击“注册”绿色按钮。

API Key注册完成后，将密匙（Key）复制，返回Ban Checker for Steam选项页面。

 

3、设置

打开Ban Checker for Steam选项，如图

在Ban Checker for Steam选项页面，将复制的密匙粘贴到输入框“Custom API key”内，并勾选“ ”，然后点击“Save”保存。

以上，准备步骤完成。

回到CSGO个人游戏数据页面，会看到导航栏多出3个按钮，点击左边的“Load whole match history”按钮加载所有记录，加载完成后再点击“Check loaded matches fo bans”按钮读取所有记录玩家的封禁状态

然后便能看到记录列表中被举报玩家的封禁状态，无需一个一个手动点头像进入资料页面查询，如图：

*红字封禁后面的+数字是指举报后多少天被封禁
