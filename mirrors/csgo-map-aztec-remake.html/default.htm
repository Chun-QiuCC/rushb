CSGO社区推出雨林遗迹/Aztec重制版

2021-02-23 分类：CSGO创意工坊 阅读(7552) 评论(0)

时隔多年，雨林遗迹/Aztec重制版终于推出，不过是社区制作，并不是V社官方。

雨林遗迹/Aztec，也就是玩家俗称的“吊桥”，反恐精英系列官方经典地图，背景设定在雨林中的阿兹特克遗迹里。从CSGO上市起，雨林遗迹就一直包含在官方休闲模式里，但由于存在严重的平衡问题，这张地图并不受广大玩家喜欢。

直到2017年九头蛇大行动结束，CSGO团队决定将冷门地图移除，用新地图代替，于是，雨林遗迹连同炙热沙城、殒命大厦直接从游戏中删除，玩家无法在官服里游玩这张地图。然而时至今日，即使越来越多玩家想念这张地图，V社似乎仍没有让雨林遗迹回归的打算。

不过在2020年12月的狂牙大行动中，V社加入了新的雨林地图：远古遗迹/Ancient。在游戏中文件，远古遗迹的模型、贴图都是位于aztec文件夹里。或许意味着远古遗迹是雨林遗迹的继承者，虽然是全新布局。并在之后，V社放出了远古遗迹的资产文件，允许社区创作者使用这些内容。

于是，万能的CSGO社区便满足了玩家们的要求，雨林遗迹/Aztec重制版面世。

雨林遗迹/Aztec重制版创意工坊地址：

https://steamcommunity.com/workshop/filedetails/?id=2395038276

雨林遗迹重制版作者是Henge，他将现有的远古遗迹风格移植到了雨林遗迹上，保留了经典布局，但更加高清、美观。新版Aztec一经推出便获得了创意工坊五星好评，如果你觉得不错，请支持作者。
