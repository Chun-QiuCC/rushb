【key_listboundkeys】CSGO查看哪些按键绑定了哪些功能

2019-07-17 分类：CSGO指令命令 / CSGO教程攻略 阅读(29041) 评论(0)

如果你哪天在CSGO中给按键绑定了功能（比如一键购买、动态组名）后不小心给忘了的话，亦或者是想给某个按键绑定多个功能，但不知原本功能的指令代码，那么可以通过控制台指令key_listboundkeys查看。

key_listboundkeys是一个可以在控制台列出玩家使用的按键及其绑定功能，包括默认键位，使用方法：在控制台输入key_listboundkeys回车即可。

*如果觉得控制台繁乱的话，可以先用clear指令清除。
