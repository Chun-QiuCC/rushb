CSGO战术探测手雷介绍

2018-12-11 分类：CSGO教程攻略 阅读(5331) 评论(0)

CSGO投掷道具之一战术探测手雷（Tactical Awareness Grenade）于2016年2月17日随野火大行动更新加入游戏，不过仅限当时的Coop守护任务中使用。在现有的官方游戏模式中，玩家无法使用战术探测手雷，不过部分创意工坊地图允许玩家使用，或者在本地/自建服务器中通过控制台开启。

顾名思义，战术探测手雷是用于探测敌人位置，玩家将其投掷出去后，战术探测手雷会附着在第一个接触平面，并在2秒钟内红光闪烁和发出嘀嘀声，随后发出声纳波，在玩家的视野中显示出3秒钟的墙后敌人轮廓和跟踪线，并且在雷达显示敌人位置。

其他：

  • 玩家只能携带一枚战术探测手雷
  • 战术探测手雷不会反弹，接触平面后就会开始生效
  • 被探测的敌人会受到类似闪光弹的效果

如果你想体验下战术探测手雷，可以通过控制台指令输入“sv_cheat 1”开启作弊模式，然后在输入“give weapon_tagrenade”获得战术探测手雷。
