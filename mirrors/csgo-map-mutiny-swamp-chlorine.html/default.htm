CSGO地图推荐：Mutiny、Swamp、Chlorine

2020-01-31 分类：CSGO创意工坊 阅读(4932) 评论(0)

最近CSGO创意工坊发布了多件优秀作品，本站将介绍玩家评选过的3张五星地图：Mutiny、Swamp、Chlorine。

Mutiny

首先是Mutiny，这张地图由TanookiSuit3、Huvaligen和JimWood联手设计制作，地图背景设定在藏有海盗宝藏的古老村庄遗迹，支持死亡竞赛和经典模式。

Mutiny地图订阅地址：

https://steamcommunity.com/workshop/filedetails/?id=1978052734

 

Swamp

顾名思义，这是一张以沼泽为主题的CSGO地图，作者是Morozov和BubkeZ，地图背景是以曾靠木材发展又没落的工业区，后被绿色植物也动物接管的大片沼泽。支持死亡竞赛和经典模式。

Swamp地图订阅地址：

https://steamcommunity.com/sharedfiles/filedetails/?id=1982602598

 

Chlorine

怀念Aztec/阿兹特克吗？Lizard与thewhaleman又一次为CSGO制作新的地图：Chlorine，以废弃关闭的阿兹台克主题水上乐园为背景，支持经典模式。

Chlorine地图订阅地址：

https://steamcommunity.com/sharedfiles/filedetails/?id=1981407062

在创意工坊页面点击绿色“订阅”按钮后Steam客户端便会自动下载，下载后即可游玩。

若无法访问创意工坊请查看解决方法：

https://www.rushb.net/steamcommunity-100-101-105-118.html

若出现“已断开连接-寻找专用服务器失败”提示，重启Steam客户端即可。
