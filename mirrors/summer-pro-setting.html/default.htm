Summer鼠标键盘外设、视频设置、准心CFG参数

2019-07-11 分类：选手外设/设置 阅读(5928) 评论(0)

以下是天禄/Tyloo俱乐部CSGO职业选手Summer蔡誉伦（外号fancy1、cy1、top0）所使用的鼠标、键盘、显示器、耳机等硬件外设、以及视频画面设置、准心（准星）和视角指令参数等：

配件外设

  • 显示器：明基XL2546（购买地址）
  • 鼠标：卓威 EC2-B DIVINA红（购买地址）
  • 鼠标垫：卓威 G-SR（购买地址）
  • 键盘：HYPERX 阿洛伊 专业版ALLOY FPS（购买地址）
  • 耳机：HYPERX 天箭 CLOUD FLIGHT（购买地址）
  • 显卡：英伟达GEFORCE RTX 2080 TI（购买地址）

鼠标设置

  • DPI：400
  • 灵敏度：2.70
  • eDPI：1080
  • HZ：1000
  • 开镜时灵敏度：0.95
  • 鼠标加速：关
  • 系统灵敏度：6
  • 原始输入：开

视频设置

  • 画面分辨率：1440×900
  • 宽高比：16:10
  • 缩放模式：拉伸
  • HZ：240

准心（准星）参数

cl_crosshairsize 2; cl_crosshairgap -3; cl_crosshairstyle 4; cl_crosshairusealpha 1; cl_crosshairthickness 1; cl_crosshair_drawoutline 0; cl_crosshairalpha 250; cl_crosshaircolor 1; cl_crosshairdot 0;

视角设置

viewmodel_fov 65; viewmodel_offset_x 2; viewmodel_offset_y 1.5; viewmodel_offset_z -1; viewmodel_presetpos 0; cl_viewmodel_shift_left_amt 1.5; cl_viewmodel_shift_right_amt 0.75; viewmodel_recoil 0; cl_righthand 1;
