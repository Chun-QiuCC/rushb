CSGO跑图研究、投掷物练习必备控制台指令命令

2019-06-10 分类：CSGO指令命令 / CSGO教程攻略 阅读(169514) 评论(0)

如果你想更加透彻熟悉地图，那么建议在单机模式下配合以下控制台指令/命令，能够方便、深入了解地图，包括研究、练习投掷物（燃烧弹、烟雾弹等）、哪些地方可以双架站人、哪些地方可以穿人打墙后的敌人等等。

首先需要打开控制台功能，进入游戏——“设置”——“游戏设置”——“游戏”——“启用开发者控制台（~）”，选择“是”，之后按“~”键（ESC下面）打开控制台功能。

打开游戏，选择“机器人练习赛”（单机），选择竞技模式和地图后开始游戏，本文以殒命大厦为例，今后如果CSGO官方加入了新地图，玩家也可以通过以下方法熟悉地图。

 

必备控制台命令

进入地图后，按“~”打开控制台，输入以下基础需要的控制台命令：

sv_cheats 1;sv_infinite_ammo 1;mp_roundtime_defuse 60;bot_stop 1;mp_drop_knife_enable 1;mp_buy_anywhere 1;mp_buytime 99999;mp_startmoney 16000;mp_restartgame 1;

命令对应的效果分别是：

  • sv_cheats 1;打开Cheat模式，后面的指令才能生效
  • sv_infinite_ammo 1;无限弹药
  • mp_roundtime_defuse 60;回合时间60分钟
  • bot_stop 1;机器人静止
  • mp_buy_anywhere 1;任何地方购买武器装备
  • mp_buytime 99999;超长购买时间
  • mp_startmoney 16000;回合初始金钱
  • mp_restartgame 1;重开比赛

之后游戏会重开，然后新的回合时间为60分钟，玩家可以尽情跑图和购买武器及投掷物。

为了防止误伤自己，输入以下控制台命令开启无敌：

gods;

gods为全体无敌，如果只要自己无敌，则输入god。

 

飞行穿墙

使用飞天功能，能够快速移动到目的地，比如扔烟后快速移动到落地点查看烟雾位置是否正确等等，建议将飞天穿墙指令绑定到按键上，以便随时快速移动，以F1为例。

bind F1 "noclip"

这样，按F1即可飞天，再按F1即可落地，你也可以换成其他按键，飞天速度设置及更多介绍请看：

《CSGO飞天穿墙指令noclip设置移动速度，及卡住不动解决方法》

 

寻找双架点

在CSGO地图中，有许多地方看着可以上但其实是禁止的，哪怕是双人/3人搭人梯。为了能够直观看出哪些地方能够上去，我们可以输入空气墙显示指令：

r_drawclipbrushes 2

这样，原本透明的空气墙会变成红色，红墙区域表示玩家无法进入，若要恢复透明，则输入：r_drawclipbrushes 0

当你发现某处没有空气墙想尝试能否双架，可输入以下机器人指令尝试：

放置Bot

bot_place

在视野前方放置一个Bot，记得要留够位置，

Bot蹲下

bot_crouch 1

Bot蹲下后，蹲下后我们可以跳到它的头上

Bot站立

bot_crouch 0

站到Bot头上后， 需要Bot恢复站姿，不过玩家需要跳一下它才会站起来。

 

研究穿墙

为了研究哪些墙面可直接穿透子弹，我们需要开启着弹点显示指令：

*威力越大的武器能穿的地方越多

sv_showimpacts 1

*本地着弹点为红色，服务器着弹点为蓝色

另外需要的话，可以修改着弹点显示时间：

sv_showimpacts_time 15

着弹点15秒后消失，你也可以修改成其他时间。

 

研究投掷物

为了能够方便研究烟雾弹、燃烧弹等投掷物，建议玩家开启投掷物轨迹，这样就能知道手雷会在哪些地方反弹：

sv_grenade_trajectory 1

为了提高轨迹线识别度，设置成点划线：

sv_grenade_trajectory_dash 1

进一步提高识别图，增加轨迹线的粗度：

sv_grenade_trajectory_thickness 1

设置轨迹线保留时间，20秒为例：

sv_grenade_trajectory_time 20

粗度和时间可以按需设置。

 

上述控制台指令可以写到一个cfg文件里，使用时exec加载即可。
