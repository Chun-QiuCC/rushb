CSGO已推出的武器箱、大行动更新时间列表【新增印花胶囊】

2023-01-06 分类：CSGO新闻资讯 阅读(16187) 评论(0)

本文用于汇总记录CSGO目前已推出的大行动和武器箱（不包括纪念包）、以及印花胶囊的更新时间，仅供玩家参考。你问下个大行动和武器箱什么时候出？除了V社没人知道。

*日期来自官博更新

大行动

CSGO大行动活动期间内，玩家可以购买激活通行证在限定时间内参与活动升级硬币，体验新地图，以及获得一系列奖励，比如稀有皮肤掉落、专属武器箱等等。更多详细介绍，请看这里。

列表包含了CSGO至今推出的所有大行动，包括更新和结束日期，活动持续天数和玩家等待天数。

2022年没有大行动。

 

武器箱

CSGO武器箱即可以开出武器皮肤/饰品的箱子，箱子通过游戏内掉落获得，需要另外购买钥匙才可开启。初期几个箱子皮肤来自V社自己设计，而后则是采纳社区玩家上传至创意工坊的作品。

列表包含了CSGO至今加入的所有武器箱，包括推出/更新日期（年月日），以及两个武器箱的间隔日期，即玩家等待了多少天。

到2023年出，CSGO一共推出了38款武器箱，如果你觉得图片小，可点击放大。

印花胶囊

从2014年，CSGO到现在一共推出了33款系列贴纸印花，包括印花胶囊和多款社区印花系列、大行动兑换印花。
