CSGO游戏模式控制台指令

2020-07-12 分类：CSGO指令命令 / CSGO教程攻略 阅读(14123) 评论(0)

CSGO运营至今，官方已经推出多种游戏模式，从竞技（经典）、休闲、军备竞赛等，到后续推出的搭档、守护、头号特训等。这些游戏模式都可以直接通过控制台指令，在加载地图时直接切换至指定模式。部分地图支持多种游戏模式，同样也可以用指令实现。本教程适合单机模式或自建服务器玩家体验。

游戏模式控制台指令：

game_type <数字>;game_mode <数字>;map <地图名>;

游戏模式对应的game_type和game_mode如下：

游戏模式 game_type game_mode
休闲模式 0 0
竞技模式 0 1
搭档模式 0 2
军备竞赛 1 0
爆破模式 1 1
死亡竞赛 1 2
自定义模式 3 0
守护模式 4 0
合作精英 4 1
头号特训 6 0
练习模式 2 0

例如进入炼狱小镇的搭档模式的指令为：

game_type 0;game_mode 2;map de_inferno.bsp

注意：

只有地图支持的模式才能正常游玩，守护模式需要另外加载对应的CFG文件。
