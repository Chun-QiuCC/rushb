CSGO 暂停比赛/冻结玩家/静止移动 控制台指令

2021-09-04 分类：CSGO指令命令 阅读(15251) 评论(0)

如果你想暂停自己服务器或本地游戏当前的比赛，可通过以下控制台实现，使用后，服务器（游戏时间）会暂停，玩家无法移动、射击等操作，仅限服主使用，使用后屏幕界面中间会显示“服务器已暂停”提示。

首先，输入以下指令允许服务器使用暂停功能：

sv_pausable 1

然后再输入以下指令即可启用暂停：

pause

如果想要取消暂停，那么输入以下指令即可解除：

unpause
