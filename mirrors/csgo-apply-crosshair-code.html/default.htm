CSGO准星代码导入/应用控制台指令

2020-06-28 分类：CSGO指令命令 / CSGO教程攻略 阅读(17358) 评论(0)

如果你觉得现在CSGO自带的准星代码导入功能麻烦，我们可以直接使用准星代码导入/应用控制台指令：apply_crosshair_code，更加方便，还能用来绑定按键来切换准星。比起以前的一堆准星命令，准星代码导入指令更适合编辑CFG。

准星代码导入/应用控制台指令apply_crosshair_code使用方法，在控制台输入apply_crosshair_code 准星代码即可。

例如：apply_crosshair_code CSGO-MmNHj-byVG3-N5RkH-KaujE-bow5G

*按“~”可开启控制台，若没有请在“游戏设置”——“游戏”——“启用开发者控制台（~）”选“是”。

 

如果你想将准星绑定到按键上，控制台输入：

bind X “apply_crosshair_code CSGO-MmNHj-byVG3-N5RkH-KaujE-bow5G”

X键和准星代码可自定义。
