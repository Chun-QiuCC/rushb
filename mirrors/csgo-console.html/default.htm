CSGO：设置启用并打开控制台

2020-07-11 分类：CSGO指令命令 / CSGO教程攻略 阅读(10164) 评论(0)

本站RushB中文网至今为止已经介绍了不少CSGO控制台指令，不过其中有部分并未详细告诉玩家如何打开控制台，所以这里专门讲解如何在设置里启用控制台，以及什么按键打开控制台界面。

什么是控制台：

控制台全称开发者控制台，V社旗下游戏大部分都带有的功能，允许玩家通过控制台指令（也叫命令）自定义配置，比如准星、功能开关、按键设置等，如果你是开发人员，还能用来查看地图性能、查找BUG等。

注意，某些控制台指令可以实现类似外挂效果，比如透视，但这些指令都需要进入作弊模式，正常情况下线上服务器是无法使用的。

启用控制台

进入游戏后，点击左边栏的“齿轮图标”进入设置功能，点击“游戏设置”，在“游戏”栏的“启用开发者控制台（~）”，选则“是”即可。

打开控制台

按“~”键即可开启控制台，~键位于键盘左上角，ESC键下面就是了。注意，大部分控制台指令需要进入地图后才能使用，以下是在地图中打开控制台：
