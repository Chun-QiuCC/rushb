CSGO武器箱开箱、大行动收藏品兑换 概率

2021-09-27 分类：CSGO教程攻略 / CSGO新闻资讯 阅读(11807) 评论(0)

很老的话题，许多玩家经常开箱，都希望开出最高等级的皮肤，比如刀子或手套，或者是大行动皮肤的隐秘级皮肤。然而真正“一夜暴富”的玩家仅仅只有那些运气好的，绝大部分人都是“跑车换单车”，因为玩家大部分只能开出低品质的低端不值钱皮肤，毕竟概率已经摆在那里。

关于CSGO皮肤开出概率，包括平时的开箱，大行动商店各种收藏品兑换，亦或者胶囊的贴纸概率，都可以按照以往CSGO国服官方给的计算公式算出。除了武器箱开箱，本文还包括大行动收藏品兑换、纪念包开箱。

*非汰换合同概率。

CSGO国服公开的CS:GO随机抽取类物品概率如下：

如有几率获取极其罕见的金色特殊物品（刀或手套），则获取金色物品与隐秘物品的概率之比为2:5；

隐秘（红色）及以下品质的物品，获取本级与低一级品质物品的概率为1:5；

相同品质的不同物品（比如一个箱子有多个军规皮肤）获取概率相同；

如果能产出StatTrak™（数聚/暗金）物品，则物品为 StatTrak™（数聚/暗金）的概率为1/10。

各不同品质的名称为：

武器皮肤（从高到低）

  • 隐秘（红色）
  • 保密（粉色）
  • 受限（紫色）
  • 军规级（蓝色）
  • 工业级（浅蓝色）
  • 消费级（白灰色）

贴纸、胸章这些虽然品质名称不同，但适用上述公式。

 

武器箱各皮肤品质概率

以武器箱（纪念包除外）为例，包含从军规到隐秘到金色皮肤，按照官方给出的计算公式，我们得出：

金色: 隐秘 : 保密 : 受限 : 军规 = 2 : 5 : 25 : 125 : 625

以上不考虑暗金、磨损换算下来（小数点后五位）：

  • 军规：79.923%
  • 受限：15.985%
  • 保密：3.197%
  • 隐秘：0.639%
  • 金色：0.256%

如果考虑暗金，则除以10，比如暗金刀的概率为0.026%（不考虑皮肤）。

如果考虑相同品质不同皮肤的概率，只要除以对应的数量即可。

收藏品各皮肤品质概率

不同收藏品包含的品质不同，比如湖畔激战收藏品的收藏最高只有受限，但本文我们这里以最新推出的大行动收藏品为例，比如2021 炙热沙城 II 收藏品，包含从消费级到隐秘6个品质级别。

本段适用于大行动皮肤兑换（不包括2021列车停放站收藏品）、纪念包开箱。

隐秘 : 保密 : 受限 : 军规 : 工业 : 消费 = 1 : 5 : 25 : 125 : 625 : 3125

以上公布不考虑磨损，换算下来（约等于）：

  • 消费：80.005%
  • 工业：16.001%
  • 军规：3.200%
  • 受限：0.640%
  • 保密：0.128%
  • 隐秘：0.026%

比如，2021 炙热沙城 II 收藏品只有一把隐秘品质的AK-47 | 黄金藤蔓，那么这皮肤的概率为0.026%。

以上，就是CSGO武器箱开箱、大行动皮肤兑换的概率，但是如果你想知道指定磨损皮肤的概率，那么需要知道该皮肤所在收藏品同品质皮肤的数量，以及磨损范围，才能正确计算。
