创意工坊作品提交流程更新

2019-04-03 分类：CSGO创意工坊 / Steam 阅读(4346) 评论(0)

随着今天Steam更新，V社团队改进了创意工坊（Workshop）作品提交流程，已解决虚拟物品欺诈问题。从今天开始，首次提交的CSGO创意工坊作品需要先经过邮件验证才能公开发表。对于之前已经发表过设计作品的玩家，不会看到有任何变化。

原因

虚拟物品欺诈是什么意思？如果你经常浏览创意工坊，那么相信看到过不少带有“Free Skns”字样和高端皮肤图片的作品，表示赠送或免费获得CSGO皮肤，然而这些作品附带的链接都是危害Steam帐户的网络钓鱼链接。

更详细内容可以查看本站以前发布的《请勿轻信创意工坊的“Free Skins”（免费皮肤）》

在过去，V社的审核团队会审核玩家举报的创意工坊项目，然后会删除虚假的欺诈作品，并会联系创意工坊所有者，让他们知道自己的帐户遭到入侵。然而这种措施并不完善，应为仍有部分欺诈作品并没有被玩家举报，并出不知情的玩家视野中。

新机制

随着今日更新，Steam将向首次提交创意工坊作品的账户所有者发送电子邮件，要求他们审核并确认刚刚发布的作品。如果作品是在他们不知情的情况下发布，则可以通过电子邮件中的链接修改密码。新的提交制度目的是为了减少Steam平台的欺诈风险，而不会为内容创作者带来不必要的麻烦。

 

总的来说，V社还是原先一样“懒得管”的态度，新措施并不能阻止欺诈作品的发布，仍会有新玩家上当导致损失。
