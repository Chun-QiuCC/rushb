Dev1ce鼠标键盘外设、视频设置、准心CFG参数

2019-04-29 分类：选手外设/设置 阅读(14486) 评论(0)

以下是Ninjas in Pyjamas / NIP俱乐部CSGO职业选手Nicolai “dev1ce” Reedtz所使用的硬件外设、视频画面设置以及准心（准星）、视角指令参数等：

配件外设

  • 显示器：明基XL2546（购买地址）
  • 鼠标：卓威奇亚 EC2  （购买地址）
  • 鼠标垫：雷蛇razer Gigantus V2 巨甲虫 NiP版（购买地址）
  • 键盘：雷蛇Razer黑寡妇V3（购买地址）
  • 耳机： 雷蛇Razer 旋风黑鲨V2（购买地址）

鼠标设置

  • DPI：800
  • 灵敏度：1.00
  • eDPI：800
  • HZ：1000
  • 开镜时灵敏度：1.10
  • 鼠标加速：关
  • 系统灵敏度：6
  • 原始输入：开

视频设置

  • 画面分辨率：1024×768
  • 宽高比：4:3
  • 缩放模式：黑边
  • HZ：240

准心（准星）参数

cl_crosshairalpha 800; cl_crosshaircolor 2; cl_crosshairdot 0; cl_crosshairgap -2; cl_crosshairsize 2; cl_crosshairstyle 4; cl_crosshairusealpha 1; cl_crosshairthickness 0; cl_crosshair_drawoutline 0; cl_crosshair_sniper_width 1;

视角设置

viewmodel_fov 68; viewmodel_offset_x 2.5; viewmodel_offset_y 0; viewmodel_offset_z -2; viewmodel_presetpos 0; cl_viewmodel_shift_left_amt 0; cl_viewmodel_shift_right_amt 0; viewmodel_recoil 0; cl_righthand 1;
