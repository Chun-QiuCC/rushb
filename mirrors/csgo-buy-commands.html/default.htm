CSGO单机任意位置购买、无限购买时间控制台指令

2020-01-16 分类：CSGO指令命令 / CSGO教程攻略 阅读(89808) 评论(0)

本文将介绍如何在单机模式下或自建服务器使用购买武器、投掷物等装备的无限制控制台指令，包括任意地点购买、无限时间购买、无限购买数量以及提高金钱上限（相当于无限金钱）。

首先，在单机模式下或自建服务器进入需要的地图，然后输入作弊指令：

sv_cheats 1;

*默认按“~”键可开启控制台，如果没有请在设置里开启。

 

提高金钱上限（无限金钱）

修改该功能需要输入3条命令，首先是修改金钱上限，注意CSGO最高只能设置65535，然后就是将游戏初始金钱改为65535，之后就是游戏重开指令。

mp_maxmoney 65535;
mp_startmoney 65535;
mp_restartgame 1;

 

任意地点购买

使用后可以在地图任意位置购买武器装备，控制台输入：

mp_buy_anywhere 1;

 

无限时间购买

使用后不再有购买时间限制，可随时购买。

mp_buytime 99999;

 

武器无限数量购买

使用后同一种武器可以购买更多数量

mp_weapons_allow_typecount -1;
