CSGO新版Tuscan地图制作中

2020-09-19 分类：CSGO新闻资讯 阅读(3225) 评论(0)

去年本站曾发布过《传新版地图Tuscan制作中，几月后亮相》，这里道歉，新版Tuscan还在开发中还需要一段时间才能面世。

首先，关于Tuscan（托斯卡纳），其实早在2014年9月，地图作者Colin “Brute” Volrath就制作了CSGO版的Tuscan并上传至创意工坊供玩家下载体验，并经过半年多时间调整修改后，最后一次更新在2015年5月。

为什么这张经典地图没出现CSGO官方服务器上？

在CS时期，Tuscan原身是一张名为de_cpl_mill的地图，版权归属CPL赛事组织。后来，另一家赛事组织CEVO想在比赛中使用这张地图，但由于版权原因，便聘请了Colin “Brute” Volrath仿照de_cpl_mill制作了类似的地图，命名为de_tuscan，在之后被玩家广泛认可。

到了2014年，Brute将Tuscan移植到CSGO，并在18年发文希望V社采用，也许是由于版权问题，或是地图设计不符合V社标准，CSGO官方一直没有采用这张地图。

*de_cpl_mill版权属于CPL，de_tuscan版权属于Brute。

回到正题，新版Tuscan现在是由另一位知名地图作者catfood接手，就算你没听说过这个名字，但你肯定玩过郁金香城（Tulip）、运输货船（Shipped）、伐木工厂（Log）和红宝石酒镇（Ruby），这些地图均出自catfood之手，相信新版Tuscan也会是一张高品质地图。

在19年12月，catfood发布推文，图片为一张正在开发的CSGO地图，谁猜对就能获得一块馅饼。

眼尖的CSGO玩家马上认为这是重置的Tuscan，catfood回复猜对了，并表示正和老版Tuscan作者Brute合作，以确保完整度。

到了今年8月份，catfood再发推文：仍在尝试寻找正确的构图：

目前作者catfood只发布了以上信息，如有更多消息会告诉大家。

不过提醒下，此文并不代表V社会采用新版Tuscan，虽然catfood的高水平制作能力能够让地图符合V社标准，但决定权还是在V社手里。
