随便买，CSGO内置商店搜索物品教程

2022-09-06 分类：CSGO教程攻略 阅读(2274) 评论(0)

做CSGO内容至今，发现不少玩家不知道大部分物品（音乐盒、社区胶囊、布章等）是无限量供应的，跑去第三方市场花更多钱，让人不解，所以专门写篇教程，介绍如何在CSGO内置商店搜索需要的商品。

*CSGO内置商店主要上架社区创作的贴纸胶囊、音乐盒和V社布章包、涂鸦箱，不包括Major和大行动物品。

打开CSGO，左下角就是CSGO内置商店，一般会显示最新推出的物品，或者随机显示。

在物品栏最右边有个箭头，点击就可进入搜索界面。

比如我想买凤凰闪亮贴纸，输入凤凰即可，游戏不支持中文的话，在其他软件复制过来即可，然后点击“查找热卖物品”。

便会显示相关搜索结果，点击购买即可。
