CSGO关闭三倍显示模式方法/控制台指令

2019-11-22 分类：CSGO指令命令 / CSGO教程攻略 阅读(9366) 评论(0)

在裂网大行动更新中，CSGO团队加入了“三倍显示模式”设置，不过该功能是给多屏玩家使用，如果是单个显示器玩家启用的话会导致显示问题，如图，游戏画面宽度变为1/3，而且会导致部分分辨率玩家无法通过设置改回去（看不到设置）。

*其他情况也有可能是CSGO客户端检测到多重显示器自动启用三倍显示模式。

解决方法：

按“~”打开控制台，输入以下指令回车即可禁用：

triple_monitor_mode 0

如果你没有打开控制台功能却启用了三倍显示模式，还可以通过以下方法：

打开Steam客户端——库——找到CSGO——右键并点击“属性”

在属性窗口中点击“设置启动选项”，然后再输入“+triple_monitor_mode 0”，点击确定，之后启动游戏就正常显示了。
