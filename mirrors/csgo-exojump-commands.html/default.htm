CSGO Exo跳跃相关控制台指令

2021-08-23 分类：CSGO指令命令 阅读(6214) 评论(0)

本文将介绍CSGO头号特训装备Exo跳跃（大跳装备）的相关控制台指令，需要进入作弊模式（sv_cheats 1），可供本地模式或自建服务器使用。

装备Exo跳跃

exojump

再输入一遍可卸下装备。

Exo跳跃大跳前进加成

大跳（先蹲再跳）可跳得更远。

sv_exojump_jumpbonus_forward 0.4

默认值0.4，数值越大越远。

Exo跳跃高度加成

可跳得更高，注意跌落伤害。

sv_exojump_jumpbonus_up 0.58

默认值0.58，数值越大越高。

Exo跳跃向上冲力倍数

和上面一样可以提高跳跃高度，但向上的速度更快。

sv_jump_impulse_exojump_multiplier 1.05

默认值1.05，数值越大越高。

Exo跳跃惩罚

虽然有这条指令，但可以无视，因为玩家在落地前体力又恢复了，但可以和下面的落地惩罚配合使用。

sv_exostaminajumpcost 0.04

默认值0.04，越大体力消耗越高。

Exo跳跃落地惩罚

用于设置玩家跳跃落地后的体力消耗大小，消耗越多，需要恢复的时间更长，也就意味玩家越慢恢复正常移动速度。

sv_exostaminalandcost 0.015

默认值0.015，越大体力消耗越多。

Exo跳跃跌落伤害倍数

装备Exo跳跃时玩家从高处跌落的伤害倍数。

sv_falldamage_exojump_multiplier 0.4

默认值0.4，越大伤害越大。
