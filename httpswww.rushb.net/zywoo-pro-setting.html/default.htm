ZywOo鼠标键盘外设、视频设置、准心CFG参数

2019-07-08 分类：选手外设/设置 阅读(61896) 评论(0)

以下是Vitality俱乐部CSGO职业选手Mathieu “ZywOo” Herbaut所使用的硬件外设、视频画面设置以及准心（准星）、视角指令参数等：

配件外设

  • 显示器：明基 XL2546（购买地址）
  • 鼠标：卓威 S2 DIVINA（购买地址）
  • 鼠标垫：雷蛇 重装甲虫-宇宙-速度版-大号（购买地址）
  • 键盘：雷蛇 Huntsman Elite（购买地址）
  • 耳机：雷蛇 Kraken TE（购买地址）
  • 显卡：英伟达GEFORCE GTX 1080 TI

鼠标设置

  • DPI：400
  • 灵敏度：2.00
  • eDPI：800
  • HZ：1000
  • 开镜时灵敏度：0.80
  • 鼠标加速：关
  • 系统灵敏度：6
  • 原始输入：开

视频设置

  • 画面分辨率：1280×960
  • 宽高比：4:3
  • 缩放模式：拉伸
  • HZ：240

准心（准星）参数

cl_crosshaircolor 1; cl_crosshairdot 0; cl_crosshairgap -3; cl_crosshairsize 2; cl_crosshairstyle 4; cl_crosshairalpha 250; cl_crosshairusealpha 1; cl_crosshairthickness 0; cl_crosshair_drawoutline 0; cl_crosshair_sniper_width 1;

视角设置

viewmodel_fov 68; viewmodel_offset_x 2.5; viewmodel_offset_y 0; viewmodel_offset_z -1.5; viewmodel_presetpos 3; cl_viewmodel_shift_left_amt 1.5; cl_viewmodel_shift_right_amt 0.75; viewmodel_recoil 1; cl_righthand 1;
