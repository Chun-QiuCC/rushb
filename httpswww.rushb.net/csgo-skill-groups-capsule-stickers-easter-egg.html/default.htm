【介绍】Skill Groups段位贴纸/印花彩蛋

2018-11-16 分类：CSGO新闻资讯 阅读(8341) 评论(0)

CSGO于昨天加入了新的社区贴纸胶囊段位印花胶囊（Skill Groups Capsule），本文暂时成为段位贴纸胶囊。贴纸作者是创意工坊的daniDem，他已有不少皮肤贴纸作品被CSGO采纳。关于段位贴纸，daniDem表示，每张贴纸都可以刮出彩蛋，让我们一睹为快。

Skill Groups Capsule贴纸胶囊一共有9个段位贴纸共18张，白银和大地球贴纸有普通和闪亮版本，其余则为普通和全息版本，，部分贴纸的普通和全息版本彩蛋效果一样，所以本文只展示全息版本。

图中贴纸彩蛋通过SDK工具实现，不保证是游戏内效果，仅供参考。

白银段位贴纸

白银贴纸的普通和闪亮版本的彩蛋效果不同，普通版本刮到最后会出现“>.<”文本表情和“SORRY ABOUT THE AIM”（枪法对不住啦）。

而闪亮版本会刮剩“SLIVER”（白银）几个字，并且“STRAIGHT OUTTA”（冲出白银）也被刮掉，上面的白条则会刮出“STRAIGHT INTO”（掉进白银）。

下面的黄金到小地球段位贴纸普通和全息的彩蛋效果一致，所以只展示全息效果。

黄金段位贴纸

黄金贴纸挂到后面只保留五角星，但五角星上面会刮出“MVP”字眼

AK（大师级守卫）段位贴纸

AK段位贴纸可以把周围一圈刮掉，而且缎带会刮出“AK 47”字眼。

双AK（大师级守卫精英）段位贴纸

双AK段位比较有意思，可以把重甲T头像刮成SAS头像，而且双AK也会刮成双M4，不过SDK工具效果不理想，具体请看GIF图片

菊花（杰出大师级守卫）段位贴纸

菊花贴纸只能把周围一圈刮出，保留星星。

小老鹰（传奇之鹰）段位贴纸

小老鹰段位贴纸可以把“legendary eagle”（传奇之鹰）刮掉，而且星星可以刮出“AIM SKILL TEAMPLAY”（枪法 技术 团队配合）。

大老鹰（大师级传奇之鹰）段位贴纸

大老鹰贴纸只是保留老鹰和星星。

小地球（无上之首席大师）段位贴纸

小地球段位贴纸可以给头像刮出像素墨镜（一种流行梗，表示deal with it，已搞定），并且“SUPREME MASTER”（至高大师）会刮成“SUPREME FRAGGER”（至高突破手），另外周围会刮出KILL图标

大地球（全球精英）段位贴纸

大地球段位贴纸的普通和闪亮彩蛋不同，普通版会刮出小鸡，羽翼也就变成鸡翅，并且AK刮成接近P90的形状。

而闪亮版的彩蛋则比较单调，只是刮掉周围。
