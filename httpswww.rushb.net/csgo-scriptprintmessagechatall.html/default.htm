CSGO恶搞指令：聊天框发送开箱、封禁提示 等彩色文本

2021-05-19 分类：CSGO指令命令 阅读(25736) 评论(0)

司马DOYO禁止抄袭。

本文将介绍一种在游戏服务器聊天框发送自定义彩色文本的方法，比如在屏幕上显示虚假的玩家开箱、游戏封禁文字提示，纯属恶搞。注意，本方法只能用在自建服务器或本地游戏里，虽然标题说是指令，但实际上是Script函数。

如果你希望别人能够看到，请使用无线电方法，具体请看这里。

 

图为恶搞开箱和封禁，请勿当真。

本教程所使用的是一种游戏自带的Script函数ScriptPrintMessageChatAll，用于向服务器中所有玩家的聊天框发送文本信息，并且内容文字可以自定义颜色。而ScriptPrintMessageChatAll可以配合script指令通过控制台发出，便可以在屏幕聊天框显示自定义内容。

比如，我们在控制台输入以下内容：

script ScriptPrintMessageChatAll(" \x02 RushB中文网：https://www.rushb.net/");

script ScriptPrintMessageChatAll(" \x01 1 \x02 2 \x03 3 \x04 4 \x05 5 \x06 6 \x07 7 \x08 8 \x09 9 \x10 10 \x0A 0A \x0B 0B \x0C 0C \x0D 0D \x0E 0E \x0F 0F");

然后屏幕便会显示如图内容：

司马DOYO禁止抄袭。

从图片可以看到，文字的颜色由“\x??”决定，分别是

  • \x01：白色
  • \x02：红色
  • \x03：淡紫色
  • \x04：绿色
  • \x05：黄绿色
  • \x06：浅绿色
  • \x07：浅红色
  • \x08：灰色
  • \x09：黄色
  • \x10：橘色
  • \x0A：浅灰色
  • \x0B：浅蓝色
  • \x0C：蓝色
  • \x0D：水蓝色
  • \x0E：紫色
  • \x0F：浅红色

代码格式：

比如script ScriptPrintMessageChatAll(" \x02 RushB中文网：\x0Chttps://www.rushb.net/");，括号内要有双引号，颜色代码放在文本前面，建议与文本之间有空格隔开（一般不用，但如果是数字的话可能出错），如果颜色代码放在开头，需要先打个空格与引号隔开，否则颜色不会生效。

 

其他举例

script ScriptPrintMessageChatAll(" \x0B RushB中文网 \x01从武器箱中获得了：\x0F AWP（纪念品） | 巨虫传说");

script ScriptPrintMessageChatAll(" \x0B RushB中文网 \x01从武器箱中获得了：\x0F AK-47（StatTrak™） | 水蛇");

script ScriptPrintMessageChatAll(" \x0B RushB中文网 \x01从武器箱中获得了：\x0F M4A4（StatTrak™） | 低吟");

script ScriptPrintMessageChatAll(" \x0F RushB中文网 已经被CF:GO服务器永久封禁。");
