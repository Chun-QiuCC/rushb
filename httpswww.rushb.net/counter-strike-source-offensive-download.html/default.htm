「CS起源MOD」反恐精英：起源攻势介绍/下载/安装

2020-11-23 分类：CSGO资源壁纸 阅读(20224) 评论(0)

如果你喜欢CS起源和CSGO，不妨试试这款CSS MOD：反恐精英：起源攻势（Counter-Strike: Source Offensive）简称CSSO。这款MOD将CSGO许多游戏内容移植到CSS里，包括P2000、蝴蝶刀等各种武器（R8除外），阵营角色和探员皮肤、语音，检视动作和人物界面等等。

目前CSSO MOD为抢先体验，最新版本为0.4，还存在许多BUG，如果你感兴趣，可下载体验。

反恐精英：起源攻势MOD下载

官方下载地址：

https://www.moddb.com/mods/counter-strike-source-offensive/#7835576

百度碗盘：

链接：https://pan.baidu.com/s/1LLNv5XoDNEwMlQRC-Mbwqg
提取码：nwww

安装此MOD需要CS起源游戏本体，并会涉及到覆盖游戏文件，如果无法连接原版服务器，但可以在同MOD服务器上游玩。

详细安装方法，请查看视频：

https://www.bilibili.com/video/BV18D4y1X7qn

探员皮肤修改方法

打开控制台，如果你是CT的话，输入：

loadout_slot_agent_ct 数字

如果是T的话，输入：

loadout_slot_agent_t 数字

然后进图或重开比赛即可，不同数字对应不同探员，请自行尝试。

匕首修改方法

CT：loadout_slot_knife_weapon_ct 数字

T：loadout_slot_knife_weapon_t 数字

然后进图或重开比赛即可，不同数字对应不同匕首，请自行尝试。

同槽位武器替换方法

如果你要使用同槽位另一把武器，输入以下指令即可购买。

CT

P2000：loadout_slot_hkp2000_weapon 0

USP消音：loadout_slot_hkp2000_weapon 1

FN57：loadout_slot_fiveseven_weapon 0

CZ75：loadout_slot_fiveseven_weapon 1

M4A4：loadout_slot_m4_weapon 0

M4A1消音：loadout_slot_m4_weapon 1

MP7：loadout_slot_mp7_weapon_ct 0

MP5-SD：loadout_slot_mp7_weapon_ct 1

T

Tec9：loadout_slot_tec9_weapon 0

CZ75：loadout_slot_tec9_weapon 1

MP7：loadout_slot_mp7_weapon_t 0

MP5-SD：loadout_slot_mp7_weapon_t 1
