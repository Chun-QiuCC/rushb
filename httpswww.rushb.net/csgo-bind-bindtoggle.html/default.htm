CSGO按键绑定控制台指令教程【bind与BindToggle】

2019-10-06 分类：CSGO指令命令 / CSGO教程攻略 阅读(58667) 评论(0)

本文将教新人玩家在CSGO中如何使用按键绑定控制台指令命令等相关内容，具体教程如下：

CSGO提供了两个按键绑定指令，分别是bind与BindToggle，用于不同的控制台指令。

bind

格式：

bind x "指令"

bind指令用于将控制台指令绑定到指定按键上，一个按键可绑定多条指令（英文分号隔开），按下即生效，比如用于设置一键购买武器、装备、一键清除血迹等，例如：

按下R键换弹和清除血迹：

bind r "+reload;r_cleardecals"

按下小键盘1购买M4/AK、头甲、烟闪雷火：

bind kp_end "buy m4a1; buy hegrenade; buy flashbang; buy smokegrenade; buy incgrenade; buy vesthelm;"

 

BindToggle

格式：

BindToggle x "指令"

与biind不同，BindToggle只能绑定一条指令，主要用于部分开关指令，比如无限弹药指令：无限弹药开启（sv_infinite_ammo 1），无限弹药关闭（sv_infinite_ammo 0），需要另外输入0或1来开启关闭，这时就需要用到BindToggle，按下指定按键开启，再按下关闭。

X键控制语音通信开关

BindToggle x "voice_enable"

Q键控制左右手切换

BindToggle q "cl_righthand"

 

unbind

格式：

unbind x

如果不小心绑定成错误的按键，那么可通过unbind指令解绑。

解绑小键盘1

unbind kp_end

 

其他还有key_listboundkeys指令，用于查看所有按键绑定了什么功能，具体教程见：

https://www.rushb.net/csgo-key_listboundkeys.html
