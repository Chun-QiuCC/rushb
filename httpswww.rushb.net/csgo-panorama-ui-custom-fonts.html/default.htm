CSGO新全景UI字体修改/替换教程

2018-11-19 分类：CSGO教程攻略 阅读(11293) 评论(0)

自从CSGO使用了新的全景界面（Panorama UI）后，以前的字体修改方法已经失效，如果有需要的玩家想替换CSGO游戏字体，可以参考以下教程，本文推荐的是替换工具，无需手动修改代码，简单实用，适用Steam及国服客户端，但要注意：原则上不建议玩家修改游戏文件，若因此教程产生的问题及风险请自行承担，本站概不负责。

1、下载字体以及替换工具csgopnrmfntchngr

字体可以从各大网站下载，建议.ttf文件格式，本文以“汉仪南宫体简”中文字体为例。

字体替换工具csgopnrmfntchngr，作者为whoon-

  • 下载地址（百度网盘）：
  • 链接：https://pan.baidu.com/s/1n9cwDMz4wUbvv5kSiJE_aQ
    提取码：03b2

2、备份默认字体

找到CSGO客户端所在文件路径，将其中的\csgo\panorama\fonts文件夹复制粘贴到其他地方备用，用于以后还原字体。

3、替换字体

下载完成后，将字体替换工具csgopnrmfntchngr压缩包解压，运行其中的csgopnrmfntchngr.exe，Windows系统可能会阻止启动，点击运行即可。

在“CSGO Directory”输入CSGO客户端所在文件路径，例如：

C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive

如果不知道CSGO客户端文件路径，可通过Steam客户端——库——CSGO右键属性——本地文件——浏览本地文件，查找到。

在“Font Directory”输入要替换字体的文件所在路径

然后点击“Install”安装替换字体即可

游戏中效果如图：

如果想换回原来的字体，只需将备份的fonts覆盖到原来的路径即可。
